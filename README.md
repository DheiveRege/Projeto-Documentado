## Instalação
**Passo 1:**
            abre o git bash e execute **git init**

**Passo 2:**
            Execute **git clone + Link que te enviei** e, **cd farmacia-FQVRDCPBL**

**Passo 3:**
            Execute **npm install**


## Como usar
        Para o gerente:

**Cadastrar** 
            voce cadastra os produtos com o aparelho leitor de codigo de barra.

**Remover produto** 
            Funçao para casos que aconteça algum erro ao produto(DO SISTEMA).

**Modificar Preço** 
            Funçao utilizada para um reajuste de preço do produto.

        para o caixa:

**Modificar quantidade** 
            Funçao utilizada para um reajuste de quantidade de produtos.

**Remover da sacola**  
            Apos a leitura do codigo de barra (no caixa) se ouver um equivico na escolha do produto, esta funçao removera da sacola.

**Finalizar compra** 
            Quando finalizar a compra (Todos o historico de venda sera salvo).
